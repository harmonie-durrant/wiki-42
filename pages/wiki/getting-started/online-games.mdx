# online games

The online games are split into two parts:
- [Memory test](#memory-test)
- [Logic tests](#logic-tests)

![Games List](/images/wiki/online-games/list.png)

## Memory Test

The memory test replicates the classic memory game **[Simon](https://en.wikipedia.org/wiki/Simon_%28game%29)**

![Memory test](/images/wiki/online-games/memory.png)

## Logic tests

In this test you will give instructions to a rocket to fly to the flag allong a path. This will test your basic logic to see if you are **Born To Code** (AKA: you have a mindset that will allow you to learn to code).

![Logic test](/images/wiki/online-games/logic.png)
